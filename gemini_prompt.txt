【事故通知内容】
### 指示 ###
あなたは製品の事故原因を分析しています。
事故の通知内容から、どんな被害であったか単純に説明してください。
回答形式はJSON形式です。

### 回答形式 ###
{
	"被害":"回答"
}

### 注意事項 ###
- 指示された回答のみ出力してください。
- なるべく与えられた文章に含まれる単語のみを使用してください。
- 被害の内容だけを説明してください。

### 例 ###
事故の通知内容:
カセットこんろの五徳付プレートを洗っていたところ、指が切れ、第一関節裏側を2針縫った。

回答:
{
	"被害":"指を切った"
}

### 事故の通知内容 ###
ハンドミキサーを使用して卵と砂糖を泡立てていたところ、泡立て金具が1本破損して指にあたった。



【事故原因】
### 指示 ###
あなたは製品の事故原因を分析しています。
事故の内容から、原因を単純に説明してください。
回答形式はJSON形式です。

### 回答形式 ###
{
	"原因":"回答"
}

### 注意事項 ###
- 指示された回答のみ出力してください。
- なるべく与えられた文章に含まれる単語のみを使用してください。
- 事故の原因だけを説明してください。

### 例 ###
事故の内容:
五徳付プレートの外側の縁が処理されておらず、多少ざらついていることから、素手でプレートを洗っていた際に、外側の縁に指を強く擦りつけたため、負傷したものと推定される。なお、取扱説明書には「手、指の保護のため、必ずゴム手袋などを使用してください。」との注意が記載されている。

回答:
{
	"原因":"指を強く擦りつけた"
}

### 事故の内容 ###
事故品は、カッター下部に設置している金属製ワッシャーが曲げ部の稜線から破断していたことから、金属製ワッシャーの外注先が金型を変更した際に、曲げ半径を小さくして作成したため、曲げ部に応力が集中し、金属疲労により破損して、調理中に混入したものと推定される。
