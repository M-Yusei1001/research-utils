# 仮想環境
以下を切り替えて使用する
- **.conda : .conda** -> データ前処理など
- **.conda : pymc_env** -> pymcを用いた演算を行う。Anacondaで構築しておく。
    - インストール方法  
        https://www.pymc.io/projects/docs/en/latest/installation.html  
        Anacondaの操作は、PowerShell上で行う  
- **.devcontainer** -> MeCab用のLinux仮想環境を構築する。**5GBくらいあるので注意。**
